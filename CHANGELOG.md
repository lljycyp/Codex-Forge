# 更新日志

<!-- 只写用户侧功能相关内容 -->

## 0.2.2

- 【优化】产品名称更新为 ChatGPT Forge，统一应用内名称与界面展示
- 【优化】精简各页面顶部布局，移除重复标题栏与非必要刷新按钮，扩大有效内容展示空间

## 0.2.1

- 【优化】焕新首页与账号管理界面，集中展示当前工作环境、运行实例和账号状态，并支持更快捷地启动或维护账号
- 【修复】修正账号额度进度条的展示方向和状态颜色，使剩余额度信息更加直观准确
- 【修复】提升 ChatGPT 登录与账号信息查询后的临时文件清理稳定性，减少因文件占用导致的操作失败

## 0.2.0

- 【新增】适配 ChatGPT 桌面客户端登录与启动流程，支持通过官方登录流程新增账号，并兼容识别 API Key 认证方式
- 【优化】增强多开隔离模式的客户端识别、关闭与复制进度提示，减少切换账号或启动多开时的等待不确定感
- 【优化】完善账号详情与备份体验，补充认证方式、额度重置与限制状态展示，并在导出备份时增加敏感信息提示

## 0.1.26

- 【优化】提升 Windows 安装包构建稳定性，减少发布包异常的情况
- 【修复】提升账号授权与令牌刷新稳定性，减少新增账号或刷新登录状态失败的情况
- 【修复】优化额度自动刷新后的页面状态同步，减少界面信息不一致的情况
